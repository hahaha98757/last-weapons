# Last Weapons
하이픽셀의 좀비즈를 위한 모드입니다.

지원 언어: [English (US)](README.md), [한국어 (대한민국)](README-ko_KR.md)

## 기능
게임에서 승리했을 때 가지고 있던 무기를 보여줍니다.
<br>가지고 있던 무기나 퍼크의 레벨을 보여줍니다.
<br>컨피그에서 자세한 설정이 가능합니다.
<br>이 기능들은 Zombies Addon에 있습니다.
